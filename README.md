# KDT
쇼핑몰 - Clodi

## 📁 프로젝트 소개
프로젝트 기간 : 25/06/02 ~ 25/06/30 1개월
사용자, 판매자, 관리자가 참여하는 쇼핑몰 시스템과 배송 플랫폼을 통합한 웹 서비스입니다.


---

## ⚙ 기술 스택

### Backend
- **언어**: Java  
- **프레임워크**: Spring Boot  
- **데이터 처리**: JPA (Java Persistence API)  
- **데이터베이스**: Oracle DB  
- **빌드 도구**: gradle

## 🔑 주요 기능

### 👤 회원
- 이메일 인증 기반 회원가입 및 로그인
- 아이디/비밀번호 찾기 (이메일 인증)
- 회원 정보 수정

### 🛒 사용자 기능
- 상품 목록/상세 조회
- 장바구니 담기/삭제
- 주문 및 결제
- 주문내역
- 마이페이지

### 🛍 판매자 기능
- 상품 등록/수정/삭제
- 판매 내역 조회

### 🛠 관리자 기능
- 회원 관리
- 후기/문의글 관리
- 챗봇 관리
